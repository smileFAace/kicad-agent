**src/kicad_agent/cli.py** -- CLI wrapper using argparse:

```python
"""CLI wrapper for kicad-agent -- run operations from the terminal."""
import argparse
import json
import sys
from pathlib import Path
```

Implement a `main()` function with argparse that supports:

1. **Positional argument**: `operation` -- A JSON string or path to a JSON file containing the operation. If the argument starts with `{`, treat as inline JSON. Otherwise, treat as a file path and read the contents.

2. **Optional flags**:
   - `--project-dir` / `-p`: Project directory (defaults to current working directory)
   - `--schema` / `-s`: Print the operation JSON Schema and exit
   - `--dry-run`: Validate the operation without executing (calls validate_operation only)
   - `--verbose` / `-v`: Print detailed output including operation details dict

3. **Execution flow**:
   - If --schema: call get_operation_schema(), print JSON, exit 0
   - Read operation input (inline JSON or file)
   - If --dry-run: call validate_operation(), print validation result, exit 0 if valid, exit 1 if invalid
   - Otherwise: call handle_operation() with the JSON and project_dir
   - Print format_result() output to stdout
   - Exit 0 if result.success, exit 1 otherwise

4. **Error handling**:
   - File not found: print error to stderr, exit 1
   - Invalid JSON file content: print parse error to stderr, exit 1
   - Any unhandled exception: print message to stderr, exit 2
