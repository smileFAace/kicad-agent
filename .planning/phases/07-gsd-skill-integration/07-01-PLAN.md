From src/kicad_agent/ops/schema.py:
```python
class Operation(BaseModel):
    root: Annotated[
        AddComponentOp | RemoveComponentOp | MoveComponentOp | ModifyPropertyOp,
        Field(discriminator="op_type"),
    ]

def get_operation_schema() -> dict:
    """Export the full JSON Schema for LLM consumption."""
    return Operation.model_json_schema()
```

Operation types and their required fields:
- AddComponentOp: op_type="add_component", target_file, library_id, reference?, value?, position{x,y,angle?}
- RemoveComponentOp: op_type="remove_component", target_file, reference
- MoveComponentOp: op_type="move_component", target_file, reference, position{x,y,angle?}
- ModifyPropertyOp: op_type="modify_property", target_file, reference, property_name, new_value

TargetFile: relative path ending in .kicad_sch, .kicad_pcb, .kicad_sym, or .kicad_mod. Rejects path traversal, absolute paths, null bytes.
