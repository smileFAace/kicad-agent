  <action>Create src/kicad_agent/context.py with three public functions and one dataclass:

**ProjectSummary dataclass** (frozen=True):
```python
@dataclass(frozen=True)
class ProjectSummary:
    project_dir: Path
    schematic_files: tuple[str, ...]   # relative paths to .kicad_sch files
    pcb_files: tuple[str, ...]         # relative paths to .kicad_pcb files
    symbol_lib_files: tuple[str, ...]  # relative paths to .kicad_sym files
    footprint_files: tuple[str, ...]   # relative paths to .kicad_mod files
    component_count: int = 0           # total across all schematics
    net_count: int = 0                 # total across all PCBs
    footprint_count: int = 0           # total across all PCBs

    @property
    def has_kicad_files(self) -> bool: ...
    @property
    def total_files(self) -> int: ...
```

**discover_kicad_files(project_dir: Path) -> ProjectSummary**:
1. Validate project_dir exists and is a directory (raise FileNotFoundError if not)
2. Recursively glob for all four KiCad file extensions: *.kicad_sch, *.kicad_pcb, *.kicad_sym, *.kicad_mod
3. Compute relative paths from project_dir for each file
4. Return ProjectSummary with file lists (as sorted tuples) and zero counts

**enrich_summary(summary: ProjectSummary) -> ProjectSummary**:
1. For each schematic file: parse with parse_schematic(), create SchematicIR, count len(ir.components), add to component_count. Use try/except around each file -- if parsing fails, skip that file and continue (log a warning via logging.warning).
2. For each PCB file: parse with parse_pcb(), extract UUIDs, create PcbIR, count len(ir.footprints) and len(ir.nets), add to footprint_count and net_count. Same error handling.
3. Return a new ProjectSummary (frozen -- create new instance) with enriched counts.

**render_project_context(project_dir: Path, enrich: bool = True) -> str**:
1. Call discover_kicad_files(project_dir)
2. If enrich=True and summary.has_kicad_files: call enrich_summary(summary)
3. Render a text summary in this format:

```
KiCad Project: {project_dir.name}
Location: {project_dir}
Files: {total} total ({schematics} schematics, {pcbs} PCBs, {symbols} symbol libs, {footprints} footprint libs)

Components: {component_count} across all schematics
Nets: {net_count} across all PCBs
Footprints: {footprint_count} placed on PCBs

Files:
  Schematics:
    - {relative_path}
  PCBs:
    - {relative_path}
  Symbol Libraries:
    - {relative_path}
  Footprint Libraries:
    - {relative_path}
```

4. If no KiCad files found, return: "No KiCad files found in {project_dir}"
5. Each section (Schematics, PCBs, etc.) is only shown if that file type has entries.

Import from parser and IR modules. Use logging.warning for parse failures, not exceptions.</action>
