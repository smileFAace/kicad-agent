    **Step 2: Create handler file `src/kicad_agent/ops/connectivity_query.py`**

    Create `handle_connectivity_query(op, ir, file_path)` function:
    - Build `NetGraph.from_pcb_ir(ir)` at the top
    - Switch on `op.query_type`:
      - `"net_stats"`: Return `graph.get_net_stats()` directly (already returns dict with total_nets, total_pads, total_connections)
      - `"connected_pads"`: Call `graph.get_connected_pads(op.net_name)`, convert each PadRef tuple to list `[footprint_ref, pad_number]` for JSON serialization, return `{"net_name": op.net_name, "pads": pads_as_lists, "count": len(pads_as_lists)}`
      - `"are_connected"`: Convert `op.source` and `op.target` lists to PadRef tuples, call `graph.are_connected(source, target)`, return `{"source": op.source, "target": op.target, "connected": bool_result}`
      - `"shortest_path"`: Convert `op.source` and `op.target` to PadRef tuples, call `graph.shortest_path(source, target)`, convert path to list of `[ref, pad]` lists, return `{"source": op.source, "target": op.target, "path": path_as_lists, "length": len(path_as_lists)}`
      - `"connected_components"`: Call `graph.get_connectivity_components()`, convert each set of PadRefs to sorted list of `[ref, pad]` lists, return `{"components": components_as_lists, "count": len(components_as_lists)}`

    IMPORTANT: The handler MUST NOT call `ir._record_mutation()` or modify the IR in any way. It reads from NetGraph which reads from IR without mutation.

    ```python
    _QUERY_HANDLERS: dict[str, Callable] = {}

    def register_query(op_type: str) -> Callable:
        def decorator(fn: Callable) -> Callable:
            _QUERY_HANDLERS[op_type] = fn
            return fn
        return decorator
    ```

    Register the handler:

    Add `_execute_query` method to OperationExecutor class. This method:
    1. Parses the PCB file (same as `_execute_pcb`): `parse_pcb(file_path)`, `extract_uuids(...)`, `PcbIR(...)`
    2. Calls handler: `details = self._dispatch_query(root.op_type, root, ir, file_path)`
    3. Returns result dict WITHOUT Transaction wrapping, WITHOUT serialization
    - No `with Transaction(...)` block
    - No `serialize_pcb(...)` call
    - No `normalize_kicad_output(...)` call
    - No `txn.commit()` call

    ```python
    def _execute_query(self, op: Operation, file_path: Path) -> dict[str, Any]:
        root = op.root
        parse_result = parse_pcb(file_path)
        uuid_map = extract_uuids(parse_result.raw_content, "pcb")
        ir = PcbIR(_parse_result=parse_result, _uuid_map=uuid_map)
        details = self._dispatch_query(root.op_type, root, ir, file_path)
        return {
            "success": True,
            "operation": root.op_type,
            "target_file": root.target_file,
            "details": details,
        }
    ```

    Add `_dispatch_query` method following same pattern as `_dispatch_pcb`:
    ```python
    def _dispatch_query(self, op_type: str, op: Any, ir: PcbIR, file_path: Path) -> dict[str, Any]:
        handler = _QUERY_HANDLERS.get(op_type)
        if handler is not None:
            return handler(op, ir, file_path)
        raise ValueError(f"Unknown query op_type: {op_type!r}")
    ```

    Wire the query path into the `execute()` method. Insert BEFORE the file extension check (line 643 area), after the create-op check and file existence check:

    ```python
    # Query operations: read-only, no Transaction, no serialization
    if root.op_type in _QUERY_HANDLERS:

    ```

    This ensures query operations hit the read-only path before any Transaction is opened.
