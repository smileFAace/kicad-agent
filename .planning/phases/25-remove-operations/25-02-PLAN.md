```python
"""Remove operations for schematic elements.

Removes wires, labels, junctions, and no-connect markers by UUID using
the list-filter pattern. Records all mutations in Transaction for audit trail.

Security (threat model):
- T-25-01: UUID must match existing element exactly -- no wildcard deletion.

Usage:
    from kicad_agent.ops.remove_ops import remove_wire, remove_label

    result = remove_wire(op, ir)
"""

from typing import Any

from kicad_agent.ops._schema_remove import (
    RemoveWireOp,
    RemoveLabelOp,
    RemoveJunctionOp,
    RemoveNoConnectOp,
)


class RemoveWireError(Exception):
    """Error raised when remove_wire operation fails."""


class RemoveLabelError(Exception):
    """Error raised when remove_label operation fails."""


class RemoveJunctionError(Exception):
    """Error raised when remove_junction operation fails."""


class RemoveNoConnectError(Exception):
    """Error raised when remove_no_connect operation fails."""
```

**remove_wire(op, ir) -> dict:**
1. Call `ir.get_wire_by_uuid(op.uuid)` to find the wire
2. If None: raise RemoveWireError(f"Wire not found: {op.uuid!r}")
3. Get the wire's two endpoints: start=(points[0].X, points[0].Y), end=(points[1].X, points[1].Y)
4. For each endpoint, check if any OTHER wire, pin, junction, or label exists at that position after removal:
   a. Call `ir.get_adjacent_wires(wire)` to find wires sharing each endpoint
   b. For the start endpoint: check if any adjacent wire shares the start point, OR if any pin/junction/label exists at start
   c. For the end endpoint: check if any adjacent wire shares the end point, OR if any pin/junction/label exists at end
5. If BOTH endpoints still have at least one connection (adjacent wire, pin, junction, or label), allow removal
6. If EITHER endpoint would be orphaned (dangling -- no pin, junction, label, or remaining wire at that position after removal): raise RemoveWireError(f"Cannot remove wire {op.uuid}: removal would leave dangling endpoint. Connected elements exist at both endpoints.")
7. Remove from graphicalItems using list-filter: `schematic.graphicalItems = [item for item in schematic.graphicalItems if item is not wire]`
8. Record mutation: `ir._record_mutation("remove_wire", {"uuid": op.uuid})`
9. Return {"uuid": op.uuid, "removed": True}

**remove_label(op, ir) -> dict:**
1. Call `ir.get_label_by_uuid(op.uuid, op.label_type)` to find the label
2. If None: raise RemoveLabelError(f"Label not found: {op.uuid!r} (type={op.label_type})")
3. Remove from the correct list based on label_type:
   - "local": `schematic.labels = [l for l in schematic.labels if l is not label]`
   - "global": `schematic.globalLabels = [l for l in schematic.globalLabels if l is not label]`
   - "hierarchical": `schematic.hierarchicalLabels = [l for l in schematic.hierarchicalLabels if l is not label]`
4. Record mutation: `ir._record_mutation("remove_label", {"uuid": op.uuid, "label_type": op.label_type})`
5. Return {"uuid": op.uuid, "label_type": op.label_type, "removed": True}

**remove_junction(op, ir) -> dict:**
1. Call `ir.get_junction_by_uuid(op.uuid)` to find the junction
2. If None: raise RemoveJunctionError(f"Junction not found: {op.uuid!r}")
3. Remove via list-filter: `schematic.junctions = [j for j in schematic.junctions if j is not junction]`
4. Record mutation: `ir._record_mutation("remove_junction", {"uuid": op.uuid})`
5. Return {"uuid": op.uuid, "removed": True}

**remove_no_connect(op, ir) -> dict:**
1. Call `ir.get_no_connect_by_uuid(op.uuid)` to find the no-connect
2. If None: raise RemoveNoConnectError(f"No-connect not found: {op.uuid!r}")
3. Remove via list-filter: `schematic.noConnects = [nc for nc in schematic.noConnects if nc is not no_connect]`
4. Record mutation: `ir._record_mutation("remove_no_connect", {"uuid": op.uuid})`
5. Return {"uuid": op.uuid, "removed": True}

IMPORTANT: All list-filter uses identity check (`is not`) not equality, consistent with remove_component.py line 60.
