    `modify_copper_zone(ir, file_path, zone_uuid, net_name, layer, clearance, min_width, priority)`:
    - Find zone by UUID: `target = next((z for z in ir.board.zones if z.tstamp == zone_uuid), None)`
    - If not found: raise ValueError
    - For each non-None parameter, update the zone property:

      - layer: update zone.layers = [new_layer]
      - clearance: update zone.clearance
      - min_width: update zone.minThickness
      - priority: update zone.priority
    - Call ir._record_mutation("modify_copper_zone", {updated fields})
    - Return {"modified": True, "zone_uuid": zone_uuid, "updated_fields": [list of field names]}

    `remove_copper_zone(ir, file_path, zone_uuid, zone_index)`:
    - Try UUID first: find zone by tstamp
    - Fall back to index if UUID not provided
    - If neither provided: raise ValueError("Must specify zone_uuid or zone_index")
    - If UUID not found: raise ValueError(f"Zone with UUID '{zone_uuid}' not found")
    - If index out of range: raise IndexError
    - Remove from board.zones list
    - Call ir._record_mutation("remove_copper_zone", {"zone_uuid": target.tstamp})
    - Return {"removed": True, "zone_uuid": target.tstamp}
